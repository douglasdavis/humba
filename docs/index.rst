humba
=====

Histogramming in Python accelerated with `numba
<http://numba.pydata.org/>`_.

This package provides functions for calculating histograms (including
weighted histograms with Poisson uncertainties) implemented with
just-in-time compiled Python code via numba. The API can fit on an
index card, please give it a read.
